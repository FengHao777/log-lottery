# 数据库测试脚本

用于清空数据库并填充测试数据。

## 功能

1. 清空所有人员和奖项数据
2. 生成 500 个随机人员数据
3. 生成 12 个奖项，每个奖项随机分配 1-5 个名额

## 使用方法

### 运行脚本

```bash
npm run seed
```

### 直接使用 tsx 运行

```bash
npx tsx scripts/seed-database.ts
```

## 配置

如需修改 API 地址，请编辑 `scripts/seed-database.ts` 中的 `API_BASE_URL` 常量：

```typescript
const API_BASE_URL = 'http://localhost:8000'
```

## 生成规则

- **人员数量**: 500 个
- **奖项数量**: 12 个
- **奖项名称**: 特等奖、一等奖、二等奖、三等奖、幸运奖、参与奖、创新奖、最佳贡献奖、优秀员工奖、团队协作奖、进步最快奖、最佳新人奖
- **部门**: 技术部、市场部、运营部、财务部、人事部、产品部、设计部、客服部
- **职位**: 员工、主管、经理、总监、VP

## 注意事项

- 此脚本会删除所有现有数据，请谨慎使用
- 确保后端服务已启动并可访问
- 确保数据库连接正常
